# Contact Book 📒
A command line contact manager built with Python.

## About
This is my second Python project built while learning Python.
It runs in the terminal and saves contacts to a CSV file so
data is never lost when the program closes.

## Features
- Add new contacts with name, phone and email
- View all saved contacts with numbering
- Search contacts by name (case insensitive)
- Delete contacts by name
- Data persists between sessions using CSV

## What I learned building this
- Object Oriented Programming — Contact class with methods
- File handling — reading and writing CSV files
- Error handling — try/except for missing files
- List comprehensions — for searching and filtering contacts
- Building a complete command line application from scratch

## How to run
python contact_book.py

## Technologies used
- Python 3
- CSV module
- OS module
